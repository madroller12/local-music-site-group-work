# Apple-Style Music Streaming Web Application

A sleek, modern, and fully responsive client-side web interface inspired by the clean aesthetics of Apple Music. This project demonstrates intermediate front-end development skills, semantic layout design, and interactive UI scripting.

## Features
- **Apple-Inspired UI:** Minimalist design with smooth transitions, precise spacing, and rounded layouts.
- **Fully Responsive:** Flawless user experience across mobile phones, tablets, and desktop displays.
- **Interactive Audio Logic:** Custom JavaScript implementation for track playback, controls, and dynamic UI updates.
- **Clean Architecture:** Well-structured and commented code for high maintainability.

## Tech Stack
- **HTML5:** Semantic structuring for better accessibility.
- **CSS3:** Flexbox/Grid layouts, custom properties (variables), and modern animations.
- **JavaScript (ES6):** Client-side application logic and audio stream handling.

## Note on Audio Files
To keep this repository lightweight and fast to clone, heavy audio tracks (`.mp3` format) have been omitted from the online repository. 

**To run the full playlist locally:**
1. Clone this repository to your machine.
2. Create a folder named `/songs` (or check the path in `script.js`).
3. Add your favorite MP3 files and ensure their filenames match the array inside the JavaScript code.
